HTML重点标签

HTML常用标签 a标签 属性 href target download rel=noopener 作用 跳转外部页面 跳转内部锚点 跳转到邮箱或电话等 a的href的取值 网址 https://baidu.com http://baidu.com //baidu.com 路径 /a/b/c：以当前目录为根目录 或者a/b/c index.html 以及 ./index.html 伪协议 javascript:代码; ：可用作什么都不做，不刷新页面 mailto邮箱 tel:手机号 id href=#xxx：进行跳转到锚点 a的target的取值 内置名字 _blank：新页面打开 __top：顶层打开 _parent：父层级打开 _self：当前页面打开（默认） 程序员命名 window的name：可让两个页面在同一个窗口打开 iframe的name a的download 作用 不是打开页面，而是下载页面 问题 不是所有浏览器都支持，尤其是手机浏览器可能不支持 iframe标签 内嵌窗口，使用很少，只有一些老系统再用

第一篇技术博客

如何用hugo搭建个人博客 安装hugo 在hugo release页面下载对应版本的安装包 解压并添加至环境变量 重启终端，hugo version检查是否正确配置 搭建博客 进入hugo官网,点击Quick Start 创建新目录 hugo new site wu585.github.io.creator 增加主题 cd wu585.github.io.creator git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke 将主题添加至配置 echo 'theme = "ananke"' >> config.toml 创建博客，并将draft的true改为false hugo new posts/第一篇博客.md 启动hugo服务器 hugo server -D 修改配置：打开config.toml文件 baseURL = "https://example.org/" languageCode = "zh-Hans" title = "小吴的博客" theme = "ananke" 创建静态页面,会生成public目录，静态页面就在此目录内 hugo -D 拷贝到git

第三篇博客

如何用hugo搭建个人博客 安装hugo 打开cmder，进入一个单独的目录 hugo new site + quickstart，（注意：quickstart指的是仓库名，都要小写），会创建对应目录 cd quickstart git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke echo 'theme = "ananke"' >> config.toml hugo new posts/第一篇博客.md hugo server -D 此时会产生一个地址，点击进入即可看到自己的博客 在外层目录新建 .gitignore 文件，填写/public/ cd public git init git add . git commit -m 在github创建新仓库，仓库名必须是用户名，比如wu585.github.io git remote add origin git@github.com:Wu585/wu585.github.io.git git push -u origin master 此时，在github仓库对应的settings选项中，可看到一个链接，即为hugo博客的对应的链接
